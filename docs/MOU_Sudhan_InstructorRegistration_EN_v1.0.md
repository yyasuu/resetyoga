- Create an account via "Sign Up" (recommended email: athmalingan123@gmail.com)

**Step 2 — Instructor Application**
- From the dashboard, select "Apply as Instructor"
- Submit the following information:
  - Full name and profile photo
  - Areas of expertise (Therapeutic Yoga, Varma Therapy, etc.)
  - Credentials (Ministry of AYUSH certification, Ph.D. NIT Karaikal, etc.)

**Step 3 — Review & Approval by Reset Yoga**
- Reset Yoga will review the application (typically within 24–48 hours)
- Upon approval, the **"Legend Instructor"** badge will be displayed on Dr. Sudhan's profile

**Step 4 — First Session Setup**
- Register available time slots
- Agree on session fees
- Conduct the first lesson
